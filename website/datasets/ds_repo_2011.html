Sourcerer Repo 2011

We provide 3 data artifacts obtained from running the Sourcerer infrastructure on a collection of 70,000+ Java projects from Apache. Google Code and Sourceforge. The artifacts are very large files, and can be downloaded by anyone with enough disk space. They are available via BTSync folders, for which hash keys are provided.

The artifacts represent the same information in different formats, after different phases of processing. Under Tutorials you can learn how to replicate these artifacts with your own dataset.

The Raw Repository

This artifacts, called sourcerer_repo_2011, takes 368Gb compressed or 433Gb uncompressed, and is a collection of 74,351 Java projects and assorted jar files collected from Google Code, SourceForge and Apache as of Summer 2011. It is also integrated in an infrastructure being built by the DARPA MUSE program, which will hopefully be available for other researchers sometime in the future.

The Extracted Facts Repository

This extracted repository, called sourcerer_repo_2011_extracted, takes 84Gb compressed or 92Gb uncompressed, and is an intermediary artifact produced from the raw dataset sourcerer_repo_2011, which contains facts (fully qualified names, metrics, etc.) about each file of each project.

The Relational Database

This database, called sourcerer_repo_2011_database, takes 24Gb as a compressed MySQL dump or 448Gb when restored, and is an artifact built from the extracted facts (sourcerer_repo_2011_extracted). This database contains the output of parsing and decomposing the source code into code entities and their relations, and it is provided to facilitate static analysis. For more information about this database please check the tab Database.
