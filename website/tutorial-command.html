Command Line Tutorial

This tutorial is designed to show how to download, install, and use the Sourcerer infrastructure to build a database of Open Source Software. The requirements for this tutorial are:

  • A Java SE Development Kit (we tested with Oracle's and OpenJDK)
  • Apache Ant ( here )
  • Eclipse ( here )
  • A Mysql server ( here )

Start by clonning the Sourcerer repository:

[sourcerer-path]$ git clone https://github.com/Mondego/Sourcerer.git

where [sourcerer-path] is the directory where you downloaded Sourcerer.
